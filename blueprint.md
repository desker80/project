
# Lotto Drawing Application

## Overview

This is a simple web application that simulates a lottery drawing. It generates and displays 6 unique random numbers between 1 and 45.

## Features

*   **Lottery Number Generation:** Generates 6 unique random numbers.
*   **Modern UI:** A clean and modern user interface with animations.
*   **Responsive Design:** The application is responsive and works on different screen sizes.

## Current Plan

*   **HTML Structure:** Create the basic HTML structure for the lottery drawing machine.
*   **CSS Styling:** Style the application with a modern and visually appealing design.
*   **JavaScript Logic:** Implement the lottery number generation and display logic.
